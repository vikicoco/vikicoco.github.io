# 注意

完成所有的配置，可以尽情根据不同账号提交拉取了

注意：

在ssh提交的时候，如果配置了同平台多账户，请使用别名进行连接

默认使用git@github.com or git@gitee.com

需要改为 git@1-github:xxxxxxxxxxx

但是注意，提交的用户名

系统<全局<本地

没有本地，没有全局，会默认使用系统

所以根据每到一个仓库就创建一个本地账户

git config --local user.name "xxx"
git config --local user.email "xxx@xxx.com"

2

或者全部都使用全局账户，不推荐

查看当前仓库的配置信息

git config --local --list
