# SpringMVC常用注解归档

# @RequestMapping

用途

处理请求映射

作用在 类(提取公共路径)/方法

参数

value

  • 用来设置要映射的请求地址

path

  • 与value属性作用一致

method

  • 设置当前方法的请求方式【GET|POST|PUT|DELETE】如请求方式不支持，会报错：405

params

  • 设置当前方法的请求参数【必须携带指定参数，才可以访问当前方法】如未携带指定参数会报错，400

headers

  • 设置当前方法的请求头信息【必须携带指定请求头】如未携带指定请求头，会报错：404

# @PathVariable

用途

将占位符中参数，注入处理请求方法的形参中，要求形参与参数名一致。

参数

value

  • 设置入参参数名

name

  • 与value属性一致

required

  • 设置当前参数是否必须入参
  • true【默认值】：必须入参，如不入参会报错：500
  • false：必须入参，如不入参不会报错，会将null值注入

# RESTful风格

增 POST

删 DELETE

改 PUT

查 GET

# @RequestParam

用途

当参数名与处理请求方法中形参名一致时，SpringMVC可以实现自动入参

参数

value

  • 设置入参参数名

name

  • 与value属性作用一致

required

  • true【默认值】：必须入参，如未入参会报错：400
  • false：不必须入参，如未入参，会将null值注入【建议使用包装类】

defaultValue

  • 只要未入参，就会为指定参数设置默认值

# @RequestHeader

用途

获取请求头内容

参数

value

  • 设置获取请求头参数名称

name

  • 与value作用一致

required

  • 设置当前请求头是否必须入参
  • true【默认值】：必须入参
  • false：不必须入参

defaultValue：设置默认值

  • 如有未入参请求头，则设置当前默认值

# @CookieValue

用途

获取Cookie内容

参数

value：设置获取CookieValue参数名称

name：与value作用一致

required：设置当前CookieValue是否必须入参

  • true【默认值】：必须入参
  • false：不必须入参

defaultValue：设置默认值

  • 如有未入参CookieValue，则设置当前默认值

# @RequestBody

用途

处理请求数据

参数

required：设置当前RequestBody是否必须入参

  • true【默认值】：必须入参
  • false：不必须入参

用途

处理响应数据
